Bu layihə sadə veb əsaslı valyuta çevirici tətbiqidir. İstifadəçi daxil etdiyi məbləği RUB, EUR, USD və GBP valyutaları arasında rahat şəkildə çevirə bilər.

Tətbiq istifadəçi üçün rahat interfeys təqdim edir. İstifadəçi istədiyi məbləği daxil etdikdən sonra solda və sağda yerləşən valyuta seçimlərinə klik edərək hansı valyutadan hansı valyutaya çevirmə etmək istədiyini seçə bilər. Sistem avtomatik olaraq nəticəni hesablayır və göstərir.Bundan əlavə, nəticədə nöqtədən sonra maksimum 5 rəqəm göstərmək mümkündür ki, bu da hesablamanın daha dəqiq olmasına kömək edir.

Tətbiqdə həmçinin internet bağlantısının yoxlanılması funksiyası da mövcuddur. Əgər istifadəçinin internet bağlantısı kəsilərsə, ekranda avtomatik olaraq internetin olmadığını bildirən mesaj göstərilir.

Bundan əlavə, layihədə responsive dizayn istifadə olunub. Media queries vasitəsilə tətbiq müxtəlif ekran ölçülərinə uyğun şəkildə görünür və həm kompüterdə, həm də mobil cihazlarda rahat istifadə edilə bilir.
